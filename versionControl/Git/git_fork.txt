1. Create a fork on your website.
2. git remote add X Y
   X: <local repsository name, you create here>
   Y: <git clone path of the fork>
   after add it, you can use "git remote -v" to check it exist
3. git fetch X
   X: <local repsository name>
4. git branch --track A X/Z
   A: <local branch name, you create here>
   X: <local repsository name>
   Z: <remote branch name>, you can check list by "git remote -r"
5. git check A
   A: <local branch name>
